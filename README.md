# 2024-article-api

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/des-ant/2024-article-api.git
```

Go to the project directory

```bash
  cd 2024-article-api
```

Start the server

```bash
  $ go run ./cmd/api
  Hello world!
```
